💳 Projeto Cartão de Crédito (HTML + CSS)

Este projeto consiste na criação de um cartão de crédito estilizado inspirado no modelo Mastercard, desenvolvido utilizando apenas HTML e CSS. O objetivo foi recriar a aparência de um cartão real, aplicando conceitos de design moderno e organização visual.

Foram trabalhados elementos como cores, tipografia, posicionamento e detalhes visuais para simular um cartão de crédito, incluindo número, nome do titular, data de validade e bandeira.

🔧 Tecnologias utilizadas:

HTML5
CSS3

🎯 Objetivos do projeto:

Praticar a estruturação com HTML semântico
Aprimorar estilização com CSS
Trabalhar com posicionamento (flexbox/grid)
Reproduzir um layout realista de interface

🚀 Resultado:
Um cartão de crédito moderno e visualmente fiel ao estilo Mastercard, totalmente desenvolvido com HTML e CSS, podendo ser reutilizado em projetos de interfaces financeiras ou estudos de front-end.
